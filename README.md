# 3. Hafta Veri Seti

Bu proje kapsamında oluşturulan sentetik veri seti, öğrencilerin haftalık çalışma saatleri ile sınav skorlarını temsil etmektedir. Veri seti iki temel değişkenden oluşmaktadır:
- `haftalik_calisma_saati` (saat): Öğrencilerin haftalık ortalama çalışma süresi. 
- `sinav_skoru` (puan): Öğrencilerin sınavdan aldıkları notlar.
